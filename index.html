VoIP Integration

dracht.io lets you integrate Voice-over-IP functionality into your web applications by using middleware patterns that you are already familiar with.

Carrier-class architecture

The distributed client-server architecture allows multiple web applications to connect to and share the services of one or more always-running SIP servers.

The server component is built on the powerful open source sofia sip stack.

APIs

dracht.io provides a layered set of APIs that are easy to use and extend.

All dracht.io client and server software is freely available under the MIT open source license.
