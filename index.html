What is dracht.io?

dracht.io is an express-inspired application framework designed to let node.js developers easily integrate SIP call processing features into their applications using familiar middleware patterns. 

var drachtio = require('drachtio')() ;
app.connect({host:'localhost', port: 8022, secret: 'cymru'}) ;
app.invite( function( req, res ) {
  res.send(486,  'user-agent': "Drachtio rocksz - but he's busy right now!") ;
}) ;

drachtio.io is a client-server architecture encompassing the following software components:

  • drachtio-server is a SIP server that is written in C++ and incorporates the open source sofia sip stack.
  • drachtio is the core module that node.js applications use to interact with the drachtio server to setup and manage SIP calls.
  • drachtio-session and drachtio-redis add support for stateful SIP applications.
  • drachtio-dialog provides a higher-level framework that makes it easier to manage SIP dialogs.
  • drachtio-msml provides the ability to control IP media servers using MSML.
